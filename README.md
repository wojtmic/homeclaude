# Homeclaude
Connecting Claude to your PC.

> [!WARNING]
> This IS a personal project. It's not intended as production scale. You are still free to make Issues and PRs, but this project is not a priority.

## Usage
Connects Claude to your PC via an MCP server. You will need to set up, on your own:
- `homeclaude`.`mcp-password` setting in your Keyring as a token
- Put it behind a reverse proxy, preferrably with HTTPS
- Connect it to Claude or whatever AI you use
